Создать репозиторий:
```bash
    empty Git repository in /../..
```
Состояние:
```bash
    git status
```
Создать файл
```bash
    touch a.cpp
    git status #выдаст, что файл a.cpp есть, но не отслеживается
```

```bash 
    git add a.cpp # добавить файл в индекс (есть такой файл), 
    # сохраняем в памяти гита .git/objects/ba
    git cat-file -p hash_file # выдаст содержимое файла по хешу
```

При add file добавится просто новая версия, а не изменится старая  
В index лежит по факту текущее состояние репозитория

После коммита добавляются еще файлы  
Коммит - это файл (в нем хранится то, что в log) кто, что когда коммитил  
коммит ссылается на файл, который ссылается на файлы, 
состояния репозитория(исх файлы)

```bash
    tree # дерево репозитория    
```

git status просто проходится и пишет то, что находит, 
ничего более умного, чем проходы по графам дерева

У всех коммитов, кроме первого, есть родители

Дерево ссылается не на предыдущее дерево, а на файлы, которые существуют

```bash
    checkout #приводит в соответствие с тем, что было раньше
```

refs - каталог ссылок  
&emsp;/heads  
&emsp;&emsp;master -- имя ветки ссылка на коммит

Можно просто ручками(без гита) создать ветку, с помощью создания файла

В HEAD находится ссылка на файл ветки, где мы сейчас

Если перейдем на коммит, а не на ветку, то у него не будет головы)  
будет log -- то, до чего можно дойти сверху 

```bash
    git log --graph --all --decorate -- oneline 
    #красивое представление графа коммитов
```

можно в мастер слить несколько веток за раз
```bash
    git merge branch_1 branch_2
```

Родителей у коммита кол-во слитых веток + 1

```bash
    git commit -a -m "sdfdf" #делает коммит без добавления в индекс
```

```bash
    git branch -d #не влияет на merge
```

Надо хотя бы раз git add  

fast-forward - когда не было изменений в ветке, 
в которую вливаем, поэтому можно просто ссылку переставить

```bash
    git rebase -i (hash_commita) # последнего хорошего коммита
```
Изменение последнего коммита
```bash
    git commit --amend
```
